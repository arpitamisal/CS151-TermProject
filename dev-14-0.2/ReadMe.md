# Any other instruction that users need to know:
- Launch cs151.application.Main (IntelliJ Run) to open the landing page.
- From Home, choose Define Programming Language to add a new language.
- Enter the Name (required) and click Save — it’s stored permanently in a local file (~/.cs151_languages.txt).
- All entries will be saved in a TableView, automatically sorted A–Z.
- Click Back to Home to return.
